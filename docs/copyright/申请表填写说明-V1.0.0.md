## 三、软件功能和技术特点（表单用）

本软件为小眠 AI 移动应用提供后端服务，基于 Node.js 与 Hono 框架构建 RESTful API。主要功能包括：用户认证（访客、手机号验证码、微信 OAuth）、多域数据同步、云端对话与梦境解读代理、语音合成与语音识别代理、生物雷达数据接收与查询、记梦文生图代理、能量与商城业务、管理后台 API。采用 PostgreSQL 持久化、JWT 鉴权、Docker 容器化部署与数据库迁移机制，具有模块化路由、环境变量配置、生产健康检查等技术特点。

## 四、主要模块

| 模块 | 路径 | 功能 |
|------|------|------|
| 应用入口 | src/index.ts, src/app.ts | 服务启动与路由挂载 |
| 认证 | src/routes/auth.ts, src/services/auth.ts | 登录、JWT、微信、短信 |
| 对话语音 | src/routes/ai.ts | 对话、解梦、TTS/STT |
| 雷达 | src/routes/radar.ts, src/services/radar.ts | datapost/reportpost/查询 |

| 管理端 | src/routes/admin/* | 后台用户与审计 |
| 数据库 | src/db/migrate.ts | 表结构与迁移 |
